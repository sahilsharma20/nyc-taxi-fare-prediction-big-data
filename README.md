# NYC Taxi Fare Prediction using Big Data and PySpark

## Project Overview

This project focuses on building a big data machine learning pipeline to predict NYC taxi fare amount using PySpark.

The project uses large-scale NYC Yellow Taxi trip data and applies data preprocessing, feature engineering, machine learning model training, evaluation, and dashboard visualization.

## Problem Statement

The objective is to predict taxi fare amount based on trip-related features such as trip distance, passenger count, pickup time, pickup location, drop location, and payment type.

## Tech Stack

- Python
- PySpark
- Spark MLlib
- Pandas
- Matplotlib
- Tableau
- Google Colab
- GitHub

## Machine Learning Models

The following regression models are used:

1. Linear Regression
2. Decision Tree Regressor
3. Random Forest Regressor
4. Gradient Boosted Tree Regressor

## Evaluation Metrics

Models are evaluated using:

- RMSE
- MAE
- R2 Score
